You can merge images into a wider image. The following is a section of code from the tutorial documentation.

def merge(im1: Image.Image, im2: Image.Image) -> Image.Image:
	w = im1.size[0] + im2.size[0]
	h = max(im1.size[1], im2.size[1])
	im = Image.new("RGBA", (w,h))
	
	im.paste(im1)
	im.paste(im2, (im1.size[0], 0))
	
	return im.

Images can be pasted on each other with transparency. The value 255 indicates that it is opaque, and 0 is transparent.
